Introduction To Html

HTML(Hyper Text Markup Language) is the standard markup language used in Creating the structure of Web Pages.Markup languagesare used to determine how elements are displayed on a webpage. It consists of a series of elements.Every HTML elements are embedded inside < > ,Some Common html Elements include <P>tag, <Header>tag etc. It describes the structure of a webpage. It is used to tell the browser how to display the content.HTML when combined with CSS(Cascading Style Sheets) and JS(Javascript) have become a milestone for website deveolpment. All HTML elements unlike others don't end with .txt or .doc but .html,For example

  • Index.html
  • Documentation.html
  • Landing Page.html
  • Portfolio.html

It has a simple text-based structure that's easy for beginners to learn and understand.Here's An Example:

     
<html>
   <head>
       <title>My Website</title>
   </head>
   <body>
      <p>Hello World</p>
   </body>
</html>

How to link Css to html

They are three different ways to link html to css,They include:

  1. Inline Styling
  2. External styling
  3. Internal Styling

Inline Styling

This is when you link the css by using a style attribute after the tag e.g

<p style="color:brown";>Chicken peppersoup</p>

External styling

This is when you link the css by using a link tag e.g

<link type="text/css" href="styles.css" rel="stylesheet">

Internal Styling

This is when you link the css by using a style tag within the head e.g

<style>html{color:blue;}</style>
